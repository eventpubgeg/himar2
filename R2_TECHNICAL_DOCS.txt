2️⃣ آليات البايباس
═══════════════════════════════════════════════════════════════════════

البايباس يستخدم 3 طرق رئيسية:

A. IAT Hooking (Import Address Table)
   ─────────────────────────────────
   الطريقة: استبدال عناوين الدوال في جدول الاستيراد
   
   قبل البايباس:
   ┌─────────────────────────────────┐
   │ IAT Entry: CreateFileW          │
   │ Points to: kernel32.CreateFileW │
   └─────────────────────────────────┘
            ↓ (يستدعي الدالة الأصلية)
   
   بعد البايباس:
   ┌─────────────────────────────────┐
   │ IAT Entry: CreateFileW          │
   │ Points to: HookedCreateFileW    │
   └─────────────────────────────────┘
            ↓ (يستدعي الدالة المحجوبة)

B. String Sanitization (تنظيف السلاسل)
   ──────────────────────────────────
   عند كشف أسماء محاكيات معروفة:
   
   "emulator"       → "Generic Device"
   "nox"            → "Generic Device"
   "memu"           → "Generic Device"
   "bluestacks"     → "Generic Device"
   "gameloop"       → "Generic Device"
   ...

C. File & Registry Spoofing (إخفاء الملفات والـ Registry)
   ────────────────────────────────────────────────────
   حجب المسارات المريبة:
   
   /data/nox        → ERROR_FILE_NOT_FOUND
   /data/memu       → ERROR_FILE_NOT_FOUND
   HKEY_LOCAL_MACHINE\Software\BlueStacks → ERROR_FILE_NOT_FOUND

4️⃣ IAT Hooking - التطبيق التقني
═══════════════════════════════════════════════════════════════════════

الخطوة 1: البحث عن جدول الاستيراد
────────────────────────────────

PIMAGE_DOS_HEADER pDosHeader = (PIMAGE_DOS_HEADER)baseAddr;
if (pDosHeader->e_magic == IMAGE_DOS_SIGNATURE) {
    PIMAGE_NT_HEADERS pNTHeader = (PIMAGE_NT_HEADERS)
        ((BYTE*)baseAddr + pDosHeader->e_lfanew);
}

الخطوة 2: البحث عن الدالة المستوردة
─────────────────────────────────

PIMAGE_IMPORT_DESCRIPTOR pImportDesc = 
    (PIMAGE_IMPORT_DESCRIPTOR)((BYTE*)baseAddr + 
        pNTHeader->OptionalHeader.DataDirectory
        [IMAGE_DIRECTORY_ENTRY_IMPORT].VirtualAddress);

الخطوة 3: استبدال العنوان
──────────────────────

DWORD oldProtect;
VirtualProtect(&pThunkData->u1.Function, 
    sizeof(pThunkData->u1.Function), PAGE_READWRITE, &oldProtect);
pThunkData->u1.Function = (SIZE_T)newFunc;
VirtualProtect(&pThunkData->u1.Function, 
    sizeof(pThunkData->u1.Function), oldProtect, &oldProtect);

6️⃣ Memory Patching
═══════════════════════════════════════════════════════════════════════

الخطوات:

1. البحث عن السلاسل النصية للمحاكيات في الذاكرة
2. استبدالها بـ "Generic Device" أو قيمة حقيقية
3. معالجة كل من Unicode و ANSI strings

مثال:

char str[] = "emulator";
if (ContainsEmulatorString(str)) {
    strcpy_s(str, sizeof(str), "Generic Device");
}
